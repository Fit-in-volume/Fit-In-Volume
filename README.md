# Fit-In-Volume
상품 이미지의 가로,세로,높이를 측정해주는 개발 프로젝트
# Description
- 스마트폰 앱을 이용하여 간편하게 체적 측량을 할 수 있게 구현을 하는것이 목표
- 체적 측량앱을 통한 효율적인 상품 관리 및 진열 계획을 용이하게 하는것이 목표
